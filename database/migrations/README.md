# Миграции базы данных

Эта директория предназначена для хранения SQL миграций базы данных.

## Структура

Каждая миграция должна быть названа в формате:
`YYYYMMDD_HHMMSS_description.sql`

Пример: `20241201_120000_create_initial_tables.sql`

## Применение миграций

Миграции применяются в порядке их создания (по имени файла).

## Текущая схема БД

Система использует следующие таблицы:
- `actor` - Актеры
- `author` - Авторы
- `director` - Режиссеры
- `play` - Пьесы
- `production` - Постановки
- `performance` - Спектакли
- `rehearsal` - Репетиции
- `role` - Роли
- `theatre` - Театры
- `location` - Места проведения

Связующие таблицы:
- `actor_production` - Связь актеров и постановок
- `actor_role` - Связь актеров и ролей
- `actor_rehearsal` - Связь актеров и репетиций
- `author_play` - Связь авторов и пьес
